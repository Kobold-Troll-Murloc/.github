## 🌈 Project: P.R.I.S.M

**Physics Ray-tracing Interactive Softbody simulation Module**

우리는 단순한 그래픽을 넘어, 사용자가 실시간으로 상호작용할 수 있는 '살아있는' 물리 환경을 구축합니다.

- **P**hysics: 뉴턴 역학 기반의 정교한 물리 시스템
- **R**ay-tracing: 하이브리드 패스 트레이싱을 통한 실사 렌더링
- **I**nteractive: 실시간 조작 및 즉각적인 물리 피드백
- **S**oftbody: 유연한 물체의 변형과 충돌을 다루는 고난도 시뮬레이션
- **M**odule: 엔진 독립적인 설계를 통한 강력한 확장성

## 🛠 Tech Stack & Focus
- **Language**: Modern C++ (C++17/20)
- **Graphics Engine**: [OgreNext](https://github.com/OGRE-Next/ogre-next) (v2.3+)
- **Graphics API**: Vulkan 1.3, Direct3D11, HLMS, GLSL (SPIR-V)
- **Build System**: CMake, vcpkg, PowerShell Automation
- **Key Tech**: Hybrid Path Tracing Accumulation, Disney BSDF, PBR
